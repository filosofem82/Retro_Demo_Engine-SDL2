Pascal Retro Demo Engine
------------------------

Pascal-based demo engine that emulates the traditional Mode 13h (320x200 256 colors) and 80x50 text mode via SDL2.

Requirements:
------------
Free Pascal (FPC)

SDL2 (Development libraries and DLLs)

libopenmpt (For music playback)

Google:
------
